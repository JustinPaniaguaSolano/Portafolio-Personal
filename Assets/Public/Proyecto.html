Este portafolio web personal fue desarrollado utilizando HTML, CSS y Bootstrap para crear una interfaz atractiva y responsiva. El objetivo principal de este proyecto es mostrar mis habilidades, experiencia y proyectos de una manera profesional y accesible. He implementado un diseño limpio y moderno, asegurando que la navegación sea intuitiva para los visitantes. Además, he optimizado el sitio para diferentes dispositivos, garantizando una experiencia de usuario consistente tanto en computadoras de escritorio como en dispositivos móviles.

Proyecto desarrollado en Java con el objetivo de crear una aplicación para la gestión del inventario de repuestos de vehículos de la empresa AutopartesCR, cumpliendo con estándares de calidad y buenas prácticas de programación. El sistema permite registrar y administrar repuestos mediante una estructura orientada a objetos, utilizando la clase Repuesto para manejar información como nombre, marca, categoría y precios. Además, se implementó la clase CalculadoraImpuestos para el cálculo automático de impuestos y precios finales, evitando la entrada manual de datos. El proyecto fue utilizado para aplicar y reforzar conceptos de programación orientada a objetos, validaciones, encapsulamiento, manejo de clases, y automatización de procesos, asegurando una gestión eficiente y organizada del inventario.
